Core Principles

Fluid Layouts

Use percentage or flexible units to allow containers to adapt.

.container { width: min(90%, 1100px); }

Flexible Media

Images and videos scale with their containers.

img { max-width: 100%; height: auto; }

Breakpoints

Change layout at sensible viewport widths.

@media (max-width: 600px) { .grid { grid-template-columns: 1fr; } }
